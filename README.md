# Amazon Sales Dashboard - Power BI

## Project Overview
This project analyzes Amazon sales data using Power BI and provides actionable insights into sales performance, product categories, customer reviews, and profitability through interactive dashboards.

## Objectives
- Analyze overall sales performance.
- Identify top-performing products and categories.
- Track customer ratings and reviews.
- Monitor revenue trends and sales distribution.
- Generate business insights for decision-making.

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel/CSV

## Dataset Information
The dataset contains Amazon product sales information, including:
- Product Name
- Category
- Price
- Rating
- Review Count
- Sales Metrics

## Dashboard Features
- KPI Cards for Total Sales, Products, and Ratings
- Category-wise Sales Analysis
- Top Performing Products
- Customer Rating Distribution
- Sales Trend Visualization
- Interactive Filters and Slicers

## Key Insights
- Identified the highest revenue-generating product categories.
- Analyzed customer rating patterns across products.
- Compared sales performance between categories.
- Highlighted top-selling products and their contribution to revenue.
